# Legacy Getting-Started Notes

This folder contains older mission-style and duplicate getting-started pages.

They were moved out of the public canonical docs path during the `0.1` docs cleanup.

Why they still exist:
- they may still be useful as historical notes
- they may contain phrasing or examples worth reusing later

- they duplicated newer canonical guides
- they made the docs feel like a trail of decisions instead of a clean product manual
