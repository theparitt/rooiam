# Security Policy

Rooiam is an identity and access management project, so we take security reports seriously.

## Supported versions

Rooiam is early-stage (pre-1.0). Security fixes are applied to the `main` branch until the first stable release. There is no long-term-support branch yet.

## Reporting a vulnerability

**Please do not open a public GitHub issue for a security vulnerability.**

Report it privately:

- Use GitHub's [private vulnerability reporting](https://github.com/theparitt/rooiam/security/advisories/new) (Security → Report a vulnerability), or
- Email **security@rooiam.com**

Please include:

- the affected component (server, a specific frontend, an endpoint)
- steps to reproduce
- the impact (what an attacker can do)
- a suggested fix, if you have one

We will acknowledge your report as soon as we can and keep you updated on the fix. Once a fix is released, we're happy to credit you unless you prefer to stay anonymous.
